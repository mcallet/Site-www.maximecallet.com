Introduction :

The Master of Time : Prototype est un projet personnel réalisé sur UDK Kismet. L'intention de réalisation consistait à créer un contraste temporel entre le présent, un temple en ruine, et le passé, ce même temple en bon état dans une époque révolue. Voyagez d'une époque à l'autre afin d'avancer dans ce temple détruit, retrouvez les clefs cachées et échappez-vous.

Présentation du projet :

The Master of Time : Prototype est un niveau de jeu réalisé pour mettre en œuvre le contraste temporel. Certains éléments du décor sont détruits dans le présent, il faudra alors naviguer dans le passé pour emprunter le chemin. Certaines portes sont fermées dans le passé mais les clefs ne sont plus cachées dans le présent.
